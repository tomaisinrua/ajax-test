    HTML Fundamentals

    HyperText Markup Language (HTML) is a computer language devised to allow website creation. These websites can then be viewed by anyone else connected to the Internet. It is relatively easy to learn and quite powerful in what it allows you to create Along with Cascading Style Sheets (CSS) and JavaScript, HTML is part of the trinity that forms the cornerstone of every website. HTML is concerned with content. And it defines the structure of the web pages that we view in our browser every time we go to Facebook, Twitter, Amazon, or any other site. It takes its design and structural cues from print media, allowing us to create headings and paragraphs, insert images, bold and italicise text. And most importantly, it allows us to link to other web pages. This course will provide you with the foundations of the HTML language. You’ll learn how to create and organise the structure and content of web pages. And you’ll do this by becoming familiar and experimenting with the surprisingly small number of elements needed to express your ideas in the online world.

    User Centric Frontend Development

    Front end developers use HTML, CSS, and JavaScript to code the website and web app designs created by web designers. The code they write runs inside the user’s browser (as opposed to a back end developer, whose code runs on the web server). Think of it a little like this: the back end developer is like the engineer who designs and creates the systems that make a city work (electricity, water and sewer, zoning, etc.), while the front end developer is the one who lays out the streets and makes sure everything is connected properly so people can live their lives (a simplified analogy, but you get the rough idea). They’re also in charge of making sure that there are no errors or bugs on the front end, as well as making sure that the design appears as it’s supposed to across various platforms and browsers. In this module you'll learn the skills, tools and languages needed to create great web based user centered experiences.

    CSS Fundamentals

    With the growing popularity of HTML, it began to incorporate a lot of stylistic features, allowing developers to change the colours and size of the font, background colour and more. This led to a growing complexity of HTML, making it more difficult to use. In order to get around this, Tim Berners-Lee and Håkon Wium Lie decided to remove the stylistic approach from HTML, so it would remain a purely presentational language, and move the stylistic aspects into a separate language called Cascading Style Sheets (CSS) Cascading Stylesheets (CSS) allows developers to provide the visual impact for websites without having to write complex HTML. CSS doesn’t even have to be written into a .html file, and could be linked to from anywhere on the web, allowing for the sharing of style sheets, a practice that is commonly used by developers today. In this module you will learn the foundations of CSS. CSS allows you to enhance the User Experience (UX) for web pages. You’ll learn how to style and modify the elements that you’ve created in HTML pages, how to layout your elements in a balanced way. You’ll learn how work with different types of fonts, and much more.
